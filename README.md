# React Mobile UI Project

## Features
- React JS
- Mobile centered layout
- Multi-page navigation with React Router
- Clean project structure

## Run Locally

```bash
npm install
npm start
```

## Deploy
Upload this repository to GitHub and deploy using **Vercel** or **Netlify**.

Example:

GitHub Repo  
[https://github.com/yourusername/mobile-ui-react](https://github.com/leelat2005/mobile-ui-react)

Hosted Link  
https://mobile-ui-react.vercel.app
